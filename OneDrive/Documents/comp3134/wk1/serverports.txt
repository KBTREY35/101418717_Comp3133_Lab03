1. What is the definition of server port?
   A server port is a communication endpoint for a specific service or application which allowing data to be sent or received

2. What is server port range?
   Server ports range from 0 to 65535

3. How many bits does the number above represent?
   16 bits

4. State 10 common server ports:
   - 20: FTP 
   - 21: FTP 
   - 22: SSH 
   - 23: Telnet
   - 25: SMTP 
   - 53: DNS 
   - 80: HTTP 
   - 110: POP3 
   - 143: IMAP 
   - 443: HTTPS
